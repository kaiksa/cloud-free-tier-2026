# Contributing to Cloud Free Tier Comparison 2026

Thank you for your interest in contributing! This repository aims to provide accurate, up-to-date information about cloud provider free tiers.

## How to Contribute

### Reporting Issues
If you find outdated or incorrect information:
1. Check if an issue already exists
2. Create a new issue with:
   - Provider name
   - Specific information that needs updating
   - Source link if available

### Submitting Changes
1. Fork this repository
2. Create a branch for your changes
   ```bash
   git checkout -b fix/aws-pricing-update
   ```
3. Make your changes to `README.md`
4. Commit with a clear message
   ```bash
   git commit -m "Update AWS free tier storage limits"
   ```
5. Push to your fork
   ```bash
   git push origin fix/aws-pricing-update
   ```
6. Open a Pull Request

### What to Update

**High Priority:**
- Pricing changes
- Discontinued services
- New free tier offerings
- Incorrect specifications

**Medium Priority:**
- Added regions
- New services added to free tier
- Better descriptions

**Low Priority:**
- Formatting improvements
- Typo fixes
- Reorganization for clarity

## Guidelines

### Accuracy First
- Always provide a source link for major changes
- Verify information on official provider pages
- Note regional restrictions where applicable

### Consistency
- Follow the existing format
- Use consistent terminology
- Maintain the table structure

### Clarity
- Be concise but complete
- Highlight key limitations (regional, time-based, etc.)
- Use emojis consistently (üèÜ for best picks, etc.)
